# Telecom Customer Churn Prediction

## Project Overview

This project predicts whether a telecom customer is likely to churn or continue using the service.

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Data Encoding
6. Train Test Split
7. Model Building
8. Model Evaluation
9. Model Saving

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Dataset

Telecom Customer Churn Dataset
